# To-Do List App

This is a simple to-do list app built with React Native.

## Features
- Add new to-do items
- Mark items as done
- View the number of completed items

## Installation
1. Clone this repository
2. Run `npm install` to install the dependencies
3. Run `npm start` to start the app
